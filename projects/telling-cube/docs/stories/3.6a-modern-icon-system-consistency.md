### Icon Size Guidelines
```typescript

sm: 'h-4 w-4'   // 16px - inline with text
md: 'h-5 w-5'   // 20px - default for UI elements
```

## UX Rationale (from Sally, UX Expert)

Icons serve multiple critical UX functions:

1. **Visual Hierarchy**: Icons help users quickly scan and understand content structure
2. **Brand Consistency**: Unified icon system reinforces professional brand perception
3. **Cognitive Load**: Icons reduce reading time and improve information retention
4. **Emotional Connection**: Well-chosen icons create positive emotional responses
5. **Accessibility**: When paired with text, icons improve comprehension for all users

**Key Principles:**
- **Never use icons alone** - Always pair with text labels (except universally recognized icons like X for close)
- **Consistent sizing** - Same context = same size (all status icons should match)
- **Meaningful metaphors** - Icons should be immediately recognizable
- **Color reinforcement** - Use color to reinforce meaning (green=success, red=error)
