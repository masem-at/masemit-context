## Post-Production Checklist

- [ ] Trim any dead air / long pauses
- [ ] Speed up generation time (4x-6x) to keep pace
- [ ] Add subtle background music? (optional, keep quiet)
- [ ] **Add subtitles/captions:**
  - [ ] Option A: Burn into video (hardcoded) - works everywhere
  - [ ] Option B: Export SRT file for YouTube upload
  - [ ] Use CapCut, Descript, or Premiere for auto-transcription
  - [ ] Review and fix any transcription errors
  - [ ] Style: White text, black outline/shadow, bottom center
- [ ] Export at 1080p (not 4K, file size matters)
- [ ] Test playback on mobile (is text readable?)
