## Posting Strategy

### Timing
- **Best days:** Tuesday, Wednesday, Thursday
- **Best times:** 9-11am EST or 2-4pm EST (peak crypto Twitter)
- **Avoid:** Weekends, Monday mornings, Friday afternoons

### Hashtags
Use sparingly (max 2-3 per tweet, only on tweets 1 and 6):
- Primary: `#DAO` `#Web3`
- Secondary: `#DeFi` `#Governance`
- Avoid: `#Crypto` (too broad, spam-heavy)

### Thread Posting Method
1. Write all tweets in a thread composer first
2. Post tweet 1, then reply to yourself with tweet 2, etc.
3. Pin the thread to your profile immediately after posting
4. Don't edit after posting (kills engagement)

### Engagement Rules (First 2 Hours)
- Respond to EVERY reply within 30 minutes
- Like all genuine comments
- If someone asks "which DAO is 51%?" - answer honestly (it's Lido, the data is public)
- If someone is hostile - don't argue, ask a question instead
- Quote-retweet positive responses with brief commentary
